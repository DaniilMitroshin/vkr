# Черновая структура ВКР

## ВВЕДЕНИЕ

- Актуальность автоматизации выбора дисциплин.
- Объект исследования: процесс выбора и распределения студентов по дисциплинам.
- Предмет исследования: программные средства автоматизации распределения.
- Цель: разработать клиент-серверное приложение на Go для автоматизации распределения студентов по дисциплинам по выбору.
- Задачи: анализ предметной области, формирование требований, проектирование архитектуры и БД, реализация приложения, контейнеризация, тестирование.
- Практическая значимость.
