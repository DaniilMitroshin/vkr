# Выходные примеры

Файлы в этой папке сформированы из фактически реализованных возможностей приложения.

- `distribution_result.csv` — экспорт результатов в CSV.
- `distribution_result.json` — экспорт результатов в JSON.
- `registered_students.csv` — экспорт зарегистрированных студентов с выборами.
- `statement_example.docx` — пример заявления студента в DOCX.

Excel-файл не добавлен, потому что экспорт Excel в коде не реализован. PDF-файл не добавлен, потому что генерация PDF в коде не обнаружена.
